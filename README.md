# PWB 文件导出工具

适用于「纯纯写作」等软件的小工具，可用于导出其中写作的文章。

## 版本

当前版本：`v2.0.0`
**全新版本，C# 重构，欢迎体验！**

## 使用说明

1. 下载最新的 PWB 备份文件，并把后缀名改成 7z；
2. 使用 7-Zip 或其他压缩软件解压备份文件；
3. 找到解压后的文件夹，里面应该有一个后缀名为 db 的数据库文件；
4. 运行程序，按照提示操作。
